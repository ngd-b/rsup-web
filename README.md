## rsup-web

rsup工具前端页面。需要配合[rsup](https://github.com/ngd-b/rsup)使用，作为web端服务，提供交互。

- 国内用户访问
  [gitee-rsup](https://gitee.com/rsup/rsup);

可以访问github查看项目的具体信息。
[github-rsup](https://github.com/ngd-b/rsup);

## Features

- 展示当前npm项目依赖包信息。
- 展示依赖包的最新版本。
- 查看依赖包的`readme`信息。
- 可点击升级依赖到指定版本。
- ...
